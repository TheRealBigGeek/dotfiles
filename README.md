# Installation
### Download and change folder
```
cd ~
git clone https://github.com/TheRealBigGeek/dotfiles.git

cd dotfiles
```
### Run install
```
./install.sh
```

# Uninstallation

### Change to dotfiles folder
```
cd ~/dotfiles
```

### Run uninstaller
```
./uninstall.sh
```
# Updates
### Download updates
```
cd dotfiles

git pull https://github.com/TheRealBigGeek/dotfiles.git

```
